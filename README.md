# LUXE Skin - Cosmetic Dropshipping Page

A modern, single-page dropshipping store for cosmetic products with 3D hover effects, shopping cart functionality, and a sleek design.

## Features
- Product grid with 3D hover effects
- Shopping cart with add/remove functionality
- Customer reviews section
- Delivery and payment options display
- Simple 4-step buying process
- Fully responsive design
- No external libraries

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS 3D transforms)
- Vanilla JavaScript

## Live Demo
[View on GitHub Pages](https://[your-username].github.io/cosmetic-dropshipping/)

## How to Use
1. Browse products
2. Click "Add to cart" on any product
3. View cart items and total
4. Clear cart with the "clear" button

## Installation
1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy to GitHub Pages
